# Dotfiles

## Installation
```
export DOTFILES_PATH=~/.dotfiles/public
git clone https://raw.githubusercontent.com/fabburst/dotfiles/master/install.XXXzsh && chmod u+x install.XXXzsh && ./install.XXXzsh && rm install.XXXzsh
```
